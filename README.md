🚀 Welcome to Your Expo App
This is an Expo project bootstrapped with . It’s ready to help you build cross-platform apps for Android, iOS, and the web.
🛠️ Getting Started
- Install dependencies
npm install
- Start the development server
npx expo start

- You'll be presented with options to open the app in:
- Development build
- Android emulator
- iOS simulator
- Expo Go — a lightweight sandbox for quick testing
